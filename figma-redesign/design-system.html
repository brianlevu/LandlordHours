01 · Brand
Brand Identity
LandlordHours helps landlords track hours toward IRS Real Estate Professional Status (REPS). The brand is warm, approachable, and confidence-building — never intimidating or corporate.
App Name
LandlordHours
Tagline: "Track your path to tax qualification"
Platform
iOS 17+ · SwiftUI
Architecture: MVVM. One codebase, iPhone-only.
Design Language
Organic-Soft
Tiimo-inspired. Soft rounded shapes, warm colors, jelly-glass depth, DM Serif headlines. Feels like a wellness app, not a tax tool.

Icon System — Lucide + Jelly-Glass
All icons from the Lucide open-source library. Two modes: Naked Stroke (tab bar, navigation) and Jelly Container (categories, settings). Full reference: icon-dls.html

Naked Stroke
Tab bar, navigation, inline actions. Just the icon, no container. Color inherited from parent via currentColor.

09 · Philosophy
Design Principles
The "why" behind every visual decision.
01
Warm, Not Corporate
Users are landlords dealing with tax anxiety. The app should feel like a wellness tracker (Tiimo, Health app), not a spreadsheet. Cream backgrounds, rounded corners, friendly icons.
02
Clarity Over Density
Show one clear number per screen. Hero progress ring — not tables. Warm messages ("You're building momentum") instead of raw data dumps. Reduce cognitive load.
03
Respectful Monetization
Trial nudge = slim inline row, not giant FOMO banner. Expired state = gentle card saying "Your trial has ended" with a small Upgrade button. Never aggressive. Never scary.
04
Emotional Reward
Checking the dashboard should feel like checking a fitness ring you're proud of. Confetti on milestones. Encouraging messages. Color that shifts as you progress.
05
Progressive Disclosure
Don't show everything at once. Empty state → type description → AI suggests → auto-fill. Each step reveals the next. Complexity unfolds naturally.
06
System-Native Feel
SF Pro Rounded + New York. System materials for blur. Native springs for animation. The app should feel like it belongs on iOS, enhanced with our personality layer.
