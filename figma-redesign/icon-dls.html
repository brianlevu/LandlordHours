Design Language System

Icon System:
Organic-Soft

A unified icon language for LandlordHours powered by Lucide — an open-source icon library with rounded, friendly strokes. Every icon lives in a jelly-glass squircle container. White strokes on translucent color.

01 · Foundation
Visual DNA
Four principles that define how icons appear across the entire app.
01
Lucide Icons
All icons come from the Lucide library — pixel-perfect, consistent 24x24 grid, rounded caps and joins. No hand-drawn paths. One source of truth.
02
Squircle Container
Category and feature icons live inside a squircle (border-radius: 22%). Continuous curvature, softer than a rounded rect. Container takes the category color.
03
Jelly-Glass Depth
Three-layer container: colored body at 80% opacity, a linear glass highlight (white 30% top to transparent), and a colored drop-shadow. Icons float like soft candy.
04
Two Modes
Naked Stroke — tab bar, inline actions, navigation. Just the icon, no container.

Jelly Container — activity rows, category badges, settings, cards, empty states.
02 · Construction
Jelly-Glass Anatomy
The container is built with CSS, not SVG filters. Three layers create the floating, translucent depth.
=
4 · Lucide Icon — stroke="currentColor", color: white
3 · Glass Highlight — linear-gradient 170deg, white 30% → 0%
2 · Body Fill — category color, opacity 0.8
1 · Drop Shadow — box-shadow, category color at 35%, blur 18px
Container — border-radius: 22%, overflow: hidden

  rgba(0,0,0,0.04) 100%)
