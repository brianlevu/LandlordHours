# LandlordHours - Brand Guidelines

## Brand Overview

**App Name:** LandlordHours  
**Tagline:** *"Track your path to tax qualification"*  
**Version:** 1.0.0

## Logo Concept

### Primary Icon
- **Symbol:** House combined with clock/timer element
- **Meaning:** Time + Real Estate = Tax Qualification
- **Style:** Clean, minimal, modern

### Color Palette

| Color Name | Hex Code | Usage |
|------------|----------|-------|
| Navy Blue | #1E3A5F | Primary brand, headers, buttons |
| Green Accent | #34C759 | Success, progress, qualified indicators |
| Light Gray | #F2F2F7 | Cards, containers, backgrounds |
| Dark Text | #1C1C1E | Headlines, primary text |
| Secondary Text | #8E8E93 | Captions, labels |
| Warning Orange | #FF9500 | Alerts, near-goal |
| Error Red | #FF3B30 | Errors, delete actions |

### Typography

- **Headlines:** SF Pro Display Bold
- **Body:** SF Pro Text Regular
- **Captions:** SF Pro Text Regular
- **Timer Display:** SF Mono / System Rounded Bold

### App Icon Design

**Background:** Solid Navy Blue (#1E3A5F) or gradient  
**Foreground:** White house silhouette with clock hands  
**Accent:** Green checkmark when qualified

### Asset Requirements
- [ ] App Icon (1024x1024)
- [ ] Launch Screen Logo
- [ ] Tab Bar Icons (home, properties, track, reports, settings)
- [ ] Empty State Illustrations

## Messaging

### App Store Description
> Track your path to tax qualification. LandlordHours helps landlords log hours spent on rental properties toward the IRS 750-hour requirement for Real Estate Professional Status.
> 
> Perfect for rental property owners seeking tax advantages through material participation.
> 
> Features:
> • Easy timer with start/stop
> • Track multiple properties (LTR/STR)
> • REPS-qualified activity categorization
> • Real-time progress toward 750-hour goal
> • 50% rule calculator
> • Hours breakdown by property and category

### Key Messages
1. "Track your hours. Qualify with confidence."
2. "Your path to tax qualification starts here"
3. "Built for landlords, by landlords"
